# CodeContacts
Rails API for contacts app.

## setup
- `git clone https://jshumakerpruitt/ContactsAPIi`
- `cd ContactsAPI`
- `bundle install`
- `rake db:create`
- `rake db:migrate`
- `rake db:seed`
- `rails s -p 3001`

## Development
